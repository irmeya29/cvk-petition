# cvk-petition

Plateforme officielle de pétition CVK pour le retour de CVK sur CANAL+.

## Fonctionnalités

- Page publique responsive
- Formulaire de signature avec validation
- Anti-doublon par e-mail
- Compteur de signatures en direct
- Liste des dernières signatures publiques
- Pages légales

## Installation

```bash
composer install
php artisan key:generate
php artisan migrate --seed
php artisan serve
```
